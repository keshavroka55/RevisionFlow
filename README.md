# Revision Flow

A simple app to help you create notes and stay on top of your revisions. Perfect for anyone looking to learn better and keep track of their progress.

## Project Structure

- **back-end**: Node.js & Express API with Prisma.
- **client**: React frontend for the user dashboard.

## How to run

1. Go to the `back-end` folder and run `npm start`.
2. Go to the `client` folder and run `npm start`.
